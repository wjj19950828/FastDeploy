# FastDeploy C++ SDK

FastDeploy provides prebuilt CPP deployment libraries on Windows/Linux/Mac. Developers can download and use it directly or compile the code manually.

## Dependencies

- cuda >= 11.2
- cudnn >= 8.0
- g++ >= 5.4 (8.2 is recommended)
